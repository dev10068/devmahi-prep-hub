# 📚 Devmahi Prep Hub: SSC CGL & RRB NTPC Tracker

Welcome to the official repository of the **Devmahi Prep Hub**! 

This is a Progressive Web App (PWA) meticulously designed to help government job aspirants track the syllabus, strategy, and topic-wise weightage for the SSC CGL and RRB NTPC examinations.

## ✨ Key Features
* **📊 Topic-Wise Weightage:** Detailed breakdown of important topics and expected question frequency to help prioritize studying.
* **📱 Progressive Web App (PWA):** Installable directly on your mobile device—no app store required.
* **⚡ 100% Offline Support:** Powered by Service Workers, allowing you to access the syllabus anytime, anywhere without an active internet connection.
* **🎯 Clean UI/UX:** A minimal, fast, and responsive design tailored for mobile users.

## 🚀 Technologies Used
* **Frontend:** HTML5, CSS3, Vanilla JavaScript
* **PWA Core:** Service Workers, Web App Manifest (`manifest.json`)
* **Deployment:** GitHub Pages

## 🛠️ How to Install on Mobile
1. Open the Live App Link: [Devmahi Prep Hub](https://YOUR_GITHUB_USERNAME.github.io/Devmahi-prep-hub/) *(Note: Replace YOUR_GITHUB_USERNAME with your actual GitHub username)*
2. Open the link in Google Chrome on your Android device.
3. Tap on the three-dot menu `⋮` in the top-right corner.
4. Select **"Add to Home Screen"**.
5. The app is now installed! You can launch it directly from your app drawer and use it offline.
