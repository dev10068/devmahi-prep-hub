# 📚 Devmahi Prep Hub — SSC CGL & RRB NTPC

🔗 **Live App:** https://dev10068.github.io/devmahi-prep-hub/

A free, offline-ready PWA for SSC CGL & RRB NTPC aspirants.
No login. No ads. Just pure preparation.

## ✨ Features
- 📝 2500+ Practice Questions (SSC CGL + RRB NTPC)
- 📅 5 Years Previous Year Questions (PYQ)
- ➗ Maths Booster Section
- 🎯 Mock Tests
- 📆 Daily Planner
- 🔥 Streak System. 
- 📱 Installable PWA — works fully offline
- ⚡ 100% Free, No Ads

## 🛠️ Tech Used
HTML5 | CSS3 | Vanilla JavaScript | Service Workers | GitHub Pages

## 📲 Install on Mobile
1. Open live link in Chrome
2. Tap ⋮ menu → "Add to Home Screen"
3. Done! Works offline too.
